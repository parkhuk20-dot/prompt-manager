# 나만의 프롬프트 관리 프로그램

GenAi 프롬프트 (텍스트/이미지/영상/페르소나/자동화 등)를 카테고리, 검색, 즐겨찾기, 조회수 기준으로 관리하는 콘솔 Python 프로그램입니다.

## 실행 방법

Python 3.10 이상이 설치되어 있어야 합니다. 

\`\`\`bash
python3 prompt_manager.py
\`\`\`

실행 후 메뉴에 표시되는 번호를 입력해 원하는 기능을 선택합니다. 

## 기능 목록

1. **프롬프트 추가** - 제목, 내용, 카테고리를 입력해 새 프롬프트 등록
2. **프롬프트 목록** - 전체 프롬프트를 번호, 카테고리, 즐겨찾기 여부와 함께 출력
3. **카테고리별 조회** - 선택한 카테고리에 속한 프롬프트만 조회
4. **프롬프트 검색** - 제목 또는 내용에 포함된 키워드로 검색
5. **프롬프트 상세 보기** - 번호 선택 시 전체 내용 출력
6. **즐겨찾기 관리** - 프롬프트 번호 선택으로 즐겨찾기 추가/해제
7. **즐겨찾기 목록** - 즐겨찾기한 프롬프트만 모아서 조회

## 카테고리

- 텍스트 생성
- 이미지 생성
- 영상 생성
- 페르소나
- 자동화
- 기타

## 기본 데이터

프로그램 실행 시 이전 미션에서 작성한 프롬프트 3개가 기본으로 등록되어 있습니다.
